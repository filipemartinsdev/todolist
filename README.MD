# To Do List
> A simple app to list tasks in-terminal.
***

## About

### Features

- [x] User CRUD;
- [x] Add/Remove tasks on real-time;
- [ ] Run in anywhere (Actually localhost database);

### Stack
- [x] Java Lang
- [x] JDBC
- [x] PostgreSQL

*** 

### Requisites

- Java 17+
- PostgreSQL 16+
- Git

### Install

- `git clone https://github.com/filipemartinsdev/todolist.git`
- `cd todolist`
- working on it...
