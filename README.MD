# To Do List
> A simple app to list tasks in-terminal.
***

## About

### Features

- [x] User CRUD;
- [x] Add/Remove tasks on real-time;
- [ ] Run in anywhere (Actually localhost database);

### Stack
- [x] Java Lang
- [x] JDBC
- [x] PostgreSQL

*** 

### Requisites

- Java 17+
- PostgreSQL 14+
- Git

### Install

1- `git clone https://github.com/filipemartinsdev/todolist.git`

2- `cd todolist`

### Run
- Import PSQL Driver;
